### 패키지 인덱스 정보 업데이트
```bash
sudo apt-get update
```
### 패키지 업그레이드
```bash
sudo apt-get upgrade
```
### install package
```bash
sudo apt-get install [패키지명]
```

### remove package
```bash
sudo apt-get remove [패키지명]
```

### vim
1. remove vim-tiny
```bash
sudo apt-get remove tiny-vim
```
2. install vim
```bash
sudo apt-get install vim
```
3. vim setting
```bash
vi ~/.vimrc
```
[.vimrc](./.vimrc)

## swap file(.swp)
vim에서 문서를 편집하다가 터미널을 끄거나 다른 사용자가 해당 파일을 먼저 실행하고 있을 경우  스왑파일이 생긴다.이 스왑파일을 제거 후 다시 vim에디터로 문서를 편집할 수 있다.
