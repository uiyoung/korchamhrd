## 패키지 인덱스 정보 업데이트
```bash
sudo apt-get update
```
## 패키지 업그레이드
```bash
sudo apt-get upgrade
```
## 패키지 설치
```bash
sudo apt-get install [패키지명]
```

## 패키지 삭제
```bash
sudo apt-get remove [패키지명]
```

1. remove vim-tiny
```bash
sudo apt-get remove tiny-vim
```
2. install vim
```bash
sudo apt-get install vim
```
3. vim setting
```bash
vi ~/.vimrc
```

## swap file
vim에서 문서를  작성하다가 터미널을 끄면 스왑파일이 생긴다.
이 스왑파일을 지워줘야 다음부터 제대로 문서를 편집할 수 있다.
